## A not-so-slim-app
> Prince Mendiratta

A web app built upon Slim Framework for testing PHP and Slim concepts.

### Deployment
```git clone git@github.com:Prince-Mendiratta/not-so-slim-app.git
cd not-so-slim-app
sudo apt-get install composer (If you're on linux)
composer install
php -S localhost:8888
```
